Representação das cores

Nomes

Algumas cores têm nomes específicos em CSS, como red, blue, green, etc.

Exemplo:


p {
    color: red;
}

Hexadecimais

As cores também podem ser representadas por códigos hexadecimais, começando com o símbolo #.

Exemplo:


p {
    color: #ff0000; /* vermelho */
}

RGB

Outra forma de definir cores é usando o modelo RGB, que especifica a quantidade de vermelho, verde e azul.

Exemplo:


p {
    color: rgb(255, 0, 0); /* vermelho */
}

RGBA

O modelo RGBA é semelhante ao RGB, mas inclui um canal alfa para transparência.

Exemplo:


p {
    color: rgba(255, 0, 0, 0.5); /* vermelho com 50% de opacidade */
}

HSL

O modelo HSL define cores com base em matiz, saturação e luminosidade.

Exemplo:


p {
    color: hsl(0, 100%, 50%); /* vermelho */
}

HSLA

O modelo HSLA é semelhante ao HSL, mas também inclui um canal alfa para transparência.

Exemplo:


p {
    color: hsla(0, 100%, 50%, 0.5); /* vermelho com 50% de opacidade */
}
