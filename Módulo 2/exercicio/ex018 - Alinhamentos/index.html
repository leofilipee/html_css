Alinhamentos

Alinhamento de Texto

O alinhamento de texto pode ser feito usando a propriedade text-align.

Exemplo:


p {
    text-align: center; /* centraliza o texto */
}

Alinhamento de Imagens

As imagens podem ser alinhadas usando a propriedade float.

Exemplo:


img {
    float: right; /* alinha a imagem à direita */
}

Alinhamento de Blocos

Para alinhar blocos, como divs, você pode usar a propriedade margin.

Exemplo:


div {
    margin: 0 auto; /* centraliza o bloco */
}

Alinhamento Vertical

O alinhamento vertical pode ser feito com a propriedade vertical-align.

Exemplo:


span {
    vertical-align: middle; /* alinha o texto verticalmente ao meio */
}

Alinhamento Flexível

O modelo flexível permite alinhar itens de forma mais dinâmica.

Exemplo:


.container {
    display: flex;
    justify-content: space-between; /* distribui os itens com espaço entre eles */
}
.item {
    flex: 1; /* cada item ocupa o mesmo espaço */
}

Alinhamento de Tabelas

As tabelas podem ser alinhadas usando a propriedade text-align nas células.

Exemplo:


table {
    width: 100%;
}
td {
    text-align: right; /* alinha o conteúdo da célula à direita */
}

Alinhamento de Listas

As listas podem ser alinhadas usando a propriedade list-style-position.

Exemplo:


ul {
    list-style-position: inside; /* coloca os marcadores dentro do bloco */
}

li {
    text-align: left; /* alinha o texto da lista à esquerda */
}

Alinhamento de Botões

Os botões podem ser alinhados usando a propriedade text-align ou float.

Exemplo:


button {
    text-align: center; /* centraliza o texto do botão */
    float: right; /* alinha o botão à direita */
}
