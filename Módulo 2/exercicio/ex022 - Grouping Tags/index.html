Estrutura de um site

Nos sites, nós sempre encontraremos tags padrões que vão compor a estrutura do mesmo (header, nav, main, footer, etc).

Essas tags são chamadas de Grouping Tags e servem para agrupar outras tags dentro delas, facilitando a organização do conteúdo.

Exemplo de Grouping Tags

Veja abaixo um exemplo de como essas tags podem ser utilizadas:
