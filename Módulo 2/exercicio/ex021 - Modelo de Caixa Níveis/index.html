Modelo de Caixa : Níveis

Introdução

Este exercício demonstra os níveis de modelo de caixa do CSS, que define como os elementos HTML são exibidos na página.

Tipos de Níveis de Modelos de Caixa

O modelo de caixa do CSS pode ser dividido em dois tipos principais: box-level e inline.

Box-level

Elementos de box-level ocupam toda a largura disponível e começam em uma nova linha. Exemplos incluem <div>, <h1>, <p>.

Inline

Elementos inline ocupam apenas o espaço necessário e não iniciam uma nova linha. Exemplos incluem <span>, <a>, <strong>.
