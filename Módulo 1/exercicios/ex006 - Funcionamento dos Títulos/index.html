Título 1

Este é um exemplo de exercício para demonstrar o funcionamento dos títulos em HTML. Os títulos são importantes para a estruturação do conteúdo e para a acessibilidade do site. Eles ajudam os leitores e os motores de busca a entenderem a hierarquia e o tema das seções do documento.

Título 2

Este é um subtítulo que complementa o título principal. Os títulos de nível 2 são usados para dividir seções importantes dentro do conteúdo, permitindo uma melhor organização e leitura.

Título 3

Este é um subtítulo de nível 3, que pode ser usado para subdividir ainda mais as seções criadas pelos títulos de nível 2. É útil para detalhar tópicos específicos dentro de uma seção maior.

Título 4

Este é um subtítulo de nível 4, que pode ser usado para criar subseções dentro de um tópico já subdividido. Embora menos comum, é útil para organizar informações complexas.
