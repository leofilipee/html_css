Tipos de links

Links Externos

Você pode usar a tag <a> para criar links externos. Por exemplo, visite o W3Schools para aprender mais sobre HTML.

O "target" serve para definir onde o link será aberto. O valor "_blank" faz com que o link seja aberto em uma nova aba do navegador. Mas você também pode usar "_self" para abrir o link na mesma aba. Mas se você não especificar o "target", o link será aberto na mesma aba por padrão.

Links Internos

Você pode criar links internos para navegar dentro da mesma página. Por exemplo, Ir para o Menu Principal.

Ir para a página secreta.

Links de Download

Você pode criar links para baixar arquivos. Por exemplo, Baixar PDF. O atributo "download" indica que o link deve iniciar o download do arquivo em vez de abri-lo no navegador.
