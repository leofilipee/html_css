Parágrafos e Quebra de Linha

Tu podes escrever um parágrafo de qualquer forma. Basta colocares tudo entre as tags <p> e </p>.
A tag <br> é usada para criar uma quebra de linha. E a <hr> é usada para criar uma linha horizontal, igual à que está logo abaixo do título. Nenhuma de ambas as tags tem de ser fechadas, ou seja, não precisas de colocar </br> ou </hr>.
Para especificar no código que queres colocar a string dos "<" ou um ">", tu deves colocar "& l t ;" = < e "& g t ;" = >. Tudo isto, sem os espaços.
