A testar o uso de imagens

A ideia desta aula é aprender a inserir imagens no HTML. Para isso, vamos usar a tag <img>.

A tag <img> é usada para inserir imagens. Ela não tem uma tag de fechamento, ou seja, não precisas de colocar </img>.

O atributo "src" é usado para especificar o caminho da imagem que queres inserir. Neste caso, a imagem está no mesmo diretório do arquivo HTML.
O atributo "alt" é usado para descrever a imagem, caso ela não possa ser carregada ou para leitores de tela.

No lugar de especificar a pasta de origem da nossa image, podemos também colar diretamente no src o link/url específico para uma imagem que esteja disponível na internet. Atenção, deve ser o link da imagem, e não do site em si.
