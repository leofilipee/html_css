Negrito e Itálico

Você pode usar negrito (tag "strong") e itálico (tag "em") para destacar partes do texto.

Subscrito e Sobrescrito

Exemplo de subscrito (tag "sub") e sobrescrito (tag "sup").

Parágrafos

Os parágrafos são definidos com a tag "p". Cada parágrafo deve ser separado por uma linha em branco para melhor legibilidade.

Texto Pré-formatado

A tag "pre" no HTML é usada para renderizar texto pré-formatado. Isso significa que ela exibe o texto exatamente como ele é escrito no código-fonte, preservando espaços em branco (múltiplos espaços, quebras de linha) e a formatação original. 

        Este é um exemplo de texto pré-formatado.
        Ele preserva espaços e quebras de linha.
        Você pode escrever código ou qualquer outro texto aqui.
        Cada linha é exibida exatamente como foi escrita.

Quebras de Linha

Para inserir uma quebra de linha, você pode usar a tag "br".

Marcações

Você pode usar a tag "mark" para destacar partes importantes do texto.

Este é um exemplo de texto com marcações.

Links

Visite o Exemplo de Site para mais informações. Devemos usar a tag "a", e colocar na referência o link desejado.

Citações

"A formatação de texto é essencial para a clareza da comunicação." (tag "blockquote")

Listas

Você pode criar listas de itens usando as tags "ul" para listas não ordenadas e "ol" para listas ordenadas. Cada item da lista é definido com a tag "li".

  • Item 1
  • Item 2
  • Item 3
  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

Tabelas

As tabelas são criadas usando as tags "table", "tr" (para linhas), "th" (para cabeçalhos) e "td" (para células de dados). Aqui está um exemplo de tabela simples:

Cabeçalho 1 Cabeçalho 2
Dado 1 Dado 2
Dado 3 Dado 4

Código-fonte

Para exibir código-fonte, você pode usar a tag "code". Aqui está um exemplo:


function exemplo() {
    console.log("Este é um exemplo de código-fonte.");
}

Comentários

Você pode adicionar comentários no código HTML usando a seguinte sintaxe:

<!-- Este é um comentário -->

Conclusão

Essas são algumas das principais tags de formatação de texto em HTML. A formatação adequada do texto é fundamental para a legibilidade e compreensão do conteúdo.
