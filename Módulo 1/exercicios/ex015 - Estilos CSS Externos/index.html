Exercício 015 - Estilos CSS Externos

Este exercício demonstra como utilizar estilos CSS externos em uma página HTML.

Este parágrafo tem um estilo especial aplicado através de um arquivo CSS externo.

Para ver os estilos aplicados, abra o arquivo HTML em um navegador.

Certifique-se de que o arquivo styles.css está no mesmo diretório que este arquivo HTML.
